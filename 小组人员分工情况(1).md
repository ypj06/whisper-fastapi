## 项目背景
在短视频与长视频信息爆炸的当下，视频内容存在信息密度分散、人工梳理成本高、无法快速提炼核心知识的痛点；同时境外优质技术类、访谈类YouTube视频缺少便捷的本土化知识拆解工具，且现有通用视频转写工具存在三大短板：一是单纯语音转写仅能输出原始文稿，缺少结构化知识提炼能力；二是依赖本地GPU算力部署成本高，无法实现轻量化边缘云端部署；三是境外视频存在网络风控、字幕缺失、大模型地区访问限制等工程化落地难题。
基于上述行业痛点与现存技术缺陷，本项目搭建融合语音转写+大模型流式内容生成+边缘云部署的全链路视频知识萃取系统，分为两大核心模块：基于Faster-Whisper的本地音视频离线语音转写字幕系统、基于Cloudflare Workers+Gemini大模型的在线YouTube视频云端知识萃取系统。系统兼顾离线本地算力场景与无算力云端边缘场景，实现音视频语音识别、双语字幕生成、视频长文结构化改写、章节级5W1H结构化知识萃取全流程自动化，降低视频信息梳理门槛，同时解决大模型区域封禁、境外视频接口风控、前后端工程部署、版本托管等一系列落地难题，具备完整的工程实践价值与智能内容理解研究价值。

### 一、后端服务开发（Faster-Whisper 语音转写接口 + Cloudflare 知识萃取后端）
1. 基于`Faster-Whisper`搭建FastAPI语音转写后端，完成音视频解析、音频分片、语音识别核心逻辑；封装单文件上传、批量目录处理、模型动态加载接口，支持 mp3/mp4/mkv/wav 多格式音视频智能识别。
2. 对接百度翻译API，重构项目架构，将密钥、配置统一抽离至 `.env` 环境变量，提升项目安全性与开源规范性，适配 GitHub 开源托管标准。
3. 实现 SRT 标准字幕时间戳格式化算法，支持**原文字幕、翻译字幕双模式输出**，可按需导出 SRT 字幕文件与 TXT 纯文本文稿，完整支持批量自动化字幕生成。
4. **新增视频知识萃取后端开发**：基于 Hono + Cloudflare Workers 搭建轻量化边缘后端，完整实现 YouTube 视频字幕多路抓取、异常降级、本地字幕导入兼容逻辑；搭建 Gemini 大模型流式 SSE 生成服务，实现视频文稿增量流式输出、章节自动拆分、会话 KV 持久化存储。
5. **实现结构化知识萃取能力**：独立开发 5W1H 章节级知识解析接口，基于会话上下文复用全文内容，避免重复传参，实现单章节精准结构化提炼；适配用户自定义生成偏好（风格、受众、约束），统一全文生成与 5W1H 提炼逻辑。

### 二、前端交互页面开发（Streamlit 可视化 + 零构建 SPA 知识萃取界面）
1. 基于 Streamlit 搭建语音转写可视化前端，划分单文件上传、批量目录处理功能模块，侧边栏实现模型规格、语种选择、输出格式、覆写参数完整自定义配置。
2. 完成前后端接口联调，统一参数传递、异常捕获、结果渲染逻辑，识别完成后支持在线下载 SRT/TXT 文件，完善交互反馈与错误提示机制。
3. **新增视频知识萃取前端开发**：搭建零构建、无打包 SPA 单页应用，基于 Import Map 引入前端渲染能力，实现 Markdown 实时流式排版、章节自动渲染、5W1H 弹窗抽屉交互、全文/单章复制功能，适配视频知识萃取全流程交互。

### 三、全功能调试、迭代优化与工程部署
1. 针对语音转写模块排查并修复跨域、文件读写、百度翻译签名错误、语种映射错乱等关键 Bug，统一编码规范与参数校验逻辑，保证多语种识别稳定性。
2. 收集中英、英法、中日、德韩等多语种音频完成全链路实测，迭代优化模型识别精度、翻译适配逻辑与字幕时间戳准确性。
3. 编写 uvicorn 自动化部署脚本，解决 CUDA 加速、模型路径适配、前后端分进程启动问题，实现项目本地一键部署运行。
4. **完成 Cloudflare 云端工程化部署**：配置 Workers 边缘服务、KV 会话存储、Assets 静态资源托管，对接 AI Gateway 解决 Gemini 地区访问限制；完善健康检查、调试接口、异常降级机制，实现生产级可观测、可容错架构。

### 四、新增知识萃取核心业务贡献
1. 设计并实现「视频字幕抓取 → 大模型流式生成 → 章节拆分 → 结构化 5W1H 萃取」完整知识萃取链路，解决无官方字幕视频降级展示、网络风控自动兜底问题。
2. 搭建双模式内容输入体系：同时支持 **YouTube 在线视频解析**与**本地字幕文件导入**，彻底摆脱平台限制，拓展项目知识萃取适用场景。
3. 完成 Prompt 工程分层设计，区分固定输出规范与用户个性化偏好，保证生成内容结构统一、风格可定制，实现产品级稳定输出效果。
